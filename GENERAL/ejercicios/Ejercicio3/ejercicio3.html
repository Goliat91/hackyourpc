Introducción

En este ejercicio vamos a describir las 3 formas de añadir estilos a nuestras hojas de HTML con 3 maneras de aplicar CSS.

Aunque la forma más habituál es añadir una hoja de estilos y enlazarla desde el documento HTML, también existen otras 2 formas de aplicar estilos

  • Estilos CSS en línea.
  • Estilos CSS incrustados en la cabecera
  • Hojas de estilo externas

Estilos CSS en línea

Desde html es posible aplicar estilos individualmente a elementos html añadiendo el atributo style. Este método afectará solo al elemento en el que se integra el código.

El CSS en línea es complicado de endender y mantener ya que mezcla estilos CSS con el código HTML

Un ejemplo de Estilo CSS en línea es el siguiente:

<p style="color:green"> Párrafo de color verde.

Dejando éste resultado:

Párrafo de color verde.

Estilos CSS incrustados en la cabecera.

Otra manera sencilla de añadir estilo con CSS es con la etiqueta <Style> en la cabecera <Head> del documento HTML. Su parte negativa es que a la hora de realizar cualquier cambio, se debe realizar en las múltiples páginas diferentes y el código estará repetido.

Por ejemplo, en la hoja de éste ejercicio vamos a añadir algún atributo con la etiqueta style para cambiar el color de fondo.

<Style> * {background-color:wheat; } </Style>

Hojas de estilos externas

De una forma más práctica podemos separar el código CSS del código HTML alojándolo en una hoja independiente, en la que sólo se manipule código CSS, con referencias a los elementos, clases, ID u otro tipo de entidad hacia la hoja HTML. Esta Hoja de estilos independiente puede afectar a tantas hojas de HTML como se les haya enlazado.

El estilo se enlaza con la etiqueta <Link> en el <Head> quedando de la siguiente forma

<link rel="stylesheet" href="../CSS/Estilos.css">
