Ejemplo 04 - Mundo Suzuki ver.1

  1. Modificar el html para añadirle la hoja externa css, y añade los mismos div que en el ejercicio anterior, es decir tendrás tres divs, uno para el total, otro para el banner en el que se incluirá la imagen y otro para el contenido.
  2. Modificar la hoja de estilos, de forma que el banner no tenga el color naranja, sino que ahora sea blanco en formato hexadecimal.
  3. Eliminar la propiedad que hace que el contenido sea naranja.
  4. Añadir un selector de etiqueta para que la imagen tenga de ancho 760 pixeles, de alto 198 pixeles y tenga un contorno negro solido y de 3 pixeles.
  5. Modificar la regla h2 para que quede subrayado.
  6. Añadir una regla con el selector de etiqueta h3, de forma que el tipo de letra sea Verdana, de tamaño 20 pixeles, color azul oscuro y el espacio entre letras sea de 4 pixeles.
  7. Añadir una regla de etiqueta para modificar el estilo a la lista no ordenada de forma que la posición de los iconos de la lista sea internos, y el tipo de icono a utilizar sea cuadrado.
  8. Por último añadir en la cabecera la etiqueta necesaria para añadirle el icono a la página web.

Ejemplo 05 - Mundo Suzuki ver.2

  1. Crear un fichero externo css, y añadir la etiqueta necesaria en el fichero html.
  2. Añade un único contenedor para el contenido (agrupar el contenido con div).
  3. Incluir una regla css para el body en la que pondrás, con una única propiedad, el color de fondo #E6E6FA, la imagen de fondo "bg-page.png", y haz que dicha imagen se repita en el eje vertical y centrada
  4. Añade la regla para modificar las propiedades de todo el contenido, de forma que ocupe un ancho de 750 pixeles y esté centrado con respecto al body.
  5. Añadir una regla con un selector de etiqueta h1, para que el tipo de letra de dicha etiqueta sea Calibri, de tamaño 32 pixeles, color rojo oscuro y esté centrado.
  6. Añadir una regla con un selector de etiqueta p, para que el tipo de letra de dicha etiqueta sea Verdana, de tamaño 16 pixeles y color gris.
  7. Por último añadir en la cabecera la etiqueta necesaria para añadirle el icono a la página web.

Ejemplo 06 - Mundo Suzuki ver.3

Esta última versión va a ser una mezcla de los dos ejemplos anteriores.

  1. Copia la hoja de estilos de la primera versión de Munsdo Suzuki.
  2. Modificar la regla del body, de forma que ahora el body va a contener una única propiedad, que será la propiedad background, en la que el color será #BBB, se incluirá como imagen de fondo 'bg-page.png' y al igual que en la versión 2 ser repetirá en el eje vertical y estará centrada.
  3. Quitar la regla para el contenedor total.
  4. Añadir el icono a la página
