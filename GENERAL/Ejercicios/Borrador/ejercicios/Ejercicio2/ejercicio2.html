Selector Universal

El selector universal CSS (*) coincide con elementos de cualquier tipo, y por lo tanto todos se verán afectados por las caracteristicas aplicadas.

Selector de Etiquetas

O selectores básicos. Selecciona todos los elementos de la página cuya etiqueta HTML coincida con el valor del selector

En su ejemplo, una etiqueta pudiera ser <P> <H1> <span>, seleccionando dicha etiqueta, de forma absoluta todas se verían modificadas por la carcterística aplicada.

Un ejemplo para todas las etiquetas párrafo sería:

p {
   color: red;
}

En ocasiones, existe la posibilidad de que varias etiquetas compartan los mismos estilos. En ese caso, podemos encadenar diferentes etiquetas de la siguiente manera:

h1, h2, h3 {
   color: red;
   font-weight: normal;
   font-family: Arial, Helvetica, sans-serif; }

Selector de Clase

El selector de clase es una solución sencilla para aplicar estilos a un solo elemento de la página

Considerando la siguiente situación:

<h1>Titulo de la noticia </h1>
<p> Primer párrafo</p>
<p class="amarillo"> Segundo párrafo</p>

Claramente el párrafo afectado contiene la clase amarillo, en la hoja de estilos lo representaríamos de la siguiente manera

.amarillo {
   color: yellow;
}

Selector de ID

Algo parecido al selector de clase es el selector de ID, como diferencia, éste último solo se utiliza para un único elemento de la página. Aunque se puede utilizar para varios elementos, se considera el selector de clase más eficiente para el supuesto caso

En la hoja de HTML lo representaríamos de la siguiente forma:

<h1>Titulo de la noticia </h1>
<p> Primer párrafo</p>
<p id="importante"> Segundo párrafo</p>

Y en la hoja de estilos:

#importante {
   color: green;
}

Selector Descendiente

Este selector implica elementos que se encuentran dentro de otros elementos. Un elemento es descendiente de otro cuando se encuentra dentro de las etiquetas de apertura y cierre del otro elemento. También es posible identificar éste tipo de selector como "padre-hijo"

La morfología en html:

<h1>Titulo de la noticia </h1>
<p> Primer párrafo</p>
<a href="#">Enlace externo</a> <p> Segundo párrafo</p>

Y en hoja de estilos:

h1 p {
   color: pink;
}

Selector Adyacente

También conocido como selector hermano, aplica estilos al elemento adyacente al elemento seleccionado.

Con la siguiente estructura:

<h1>Titulo de la noticia </h1>
<p> Primer párrafo</p>
<p> Segundo párrafo</p>

Y el siguiente estilo:

h1 p {
   color: pink;
}

Y en la hoja de estilos:

h1 + p {
   font-style: italic; }

Quedando de la siguiente forma:

Título de la noticia

Primer párrafo

Segundo párrafo

Como se puede notar, el selecctor adyacente o selector hermano sólo exhibe sus cambios al adyacente siguiente al indicado. En este caso al primer párrafo adhacente del H1

Selector de Atributos

El selector de atributo nos permite aplicar estilos en base a los atributos de nuestros elementos. Un ejemplo claro se manifestaría de la siguiente forma:

<input type="user" name="" id="" placeholder="User">
<input type="password" name="" id="" placeholder="Password">

Estos dos input comparten el atributo "type=" aunque con una clase distinta. Entonces con ayuda de las llaves [] vamos a representarlo:

input[type="user"]{
  background-color:lightblue;
}
